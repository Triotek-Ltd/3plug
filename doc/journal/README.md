# 3plug Journal

## Purpose

Use this folder for state logging so we can track what we are doing, what changed, and what comes next.

## Journal Path

- `doc/journal/`

## What to Log

- Current focus / task
- Changes made
- Decisions taken
- Blockers / open questions
- Next steps
- Screenshots (SS) or references related to the session

## Suggested Naming

- Daily log: `YYYY-MM-DD.md`
- Screenshot files: `YYYY-MM-DD-topic-01.png`
- Notes with screenshots: `YYYY-MM-DD-topic.md`
